# Configs

Personal configuration files for developer tools.

## Setup

```bash
./setup.sh
```

This runs all setup scripts and first updates this repo with `git pull --ff-only`.

## What's Inside

| Directory | Tool | Description |
|-----------|------|-------------|
| `configstore/` | App configs | CLI/app state files |
| `espanso/` | [Espanso](https://espanso.org/) | Text expansion config and matches |
| `gh/` | [GitHub CLI](https://cli.github.com/) | GitHub CLI configuration |
| `git/` | Git | Shared git ignore/config snippets |
| `homebrew/` | [Homebrew](https://brew.sh/) | Brewfile and install script |
| `iterm2/` | iTerm2 | iTerm2 app support files |
| `raycast/` | [Raycast](https://www.raycast.com/) | Raycast settings and extensions |
| `starship/` | [Starship](https://starship.rs/) | Prompt configuration |

Each directory has its own README with setup instructions.
